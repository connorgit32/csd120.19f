Acronym Glossary

IP Address
A numeric address assigned to all devices connected to the internet. Consists of four groups of numbers called octets that can be a value from 0 to 255. A domanin name will bring you to that sites ip address.
HTTP
Hypertext Transfer Protocol is a set of rules for exchanging data on the web. Web browsers send HTTP requests for web pages and web servers send HTTP responses back to the browser. Invented by Tim Berners-Lee
URL
Uniform Resource Locator represents the address of a resource on the internet. It contains in order the HTTP protocol, subdomain or webserver, domain name, folder name, web page file name.
TLD
Top Level Domain identifies the right-most part of the domain name, or URL. They are used to organize domain names.
Markup Languages
Different sets of markup symbols or codes in a file intended to display on a web browser. Web browsers interpret markup languages and display it to the user. There is XHTML, XML, HTML4 but we will be using the newest one, HTML5
